Demo Form (Flask <--> jQuery <--> DataTable <--> HTML)

The purpose of this is to demonstrate the simplest possible code for displaying a form, sending the form data to flask, building demo data in flask in the DataTable format, and then displaying the DataTable in the webpage. To keep it simple, the form data is just sent back and displayed in the 'successAlert' div. The table data is hard coded into the python function and is not adjusted for the form inputs in any way. Obviously, a more useful function would use the form input to control the table data. That was left out to keep the code focused on the purpose.
